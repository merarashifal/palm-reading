# Palm Annotation Schema - Observation Tokens

Every detected feature must cite visual evidence using ONLY the following predefined observation tokens. No free-text reasoning is permitted.

## Line Evidence
- `continuous_curve`
- `strong_edge`
- `clear_origin`
- `clear_termination`
- `closed_loop`
- `ridge_intersection`
- `parallel_ridges`
- `fork_endpoint`
- `sharp_angle`
- `smooth_arc`
- `faint_trace`
- `broken_segments`

## Mount Evidence
- `raised_elevation`
- `flat_surface`
- `wide_area`
- `narrow_area`

## Sign Evidence
- `intersecting_lines`
- `star_formation`
- `triangle_formation`
- `square_formation`
- `circular_pattern`
- `island_loop`

## Skin/Texture Evidence
- `deep_creases`
- `fine_lines`
- `smooth_texture`
- `rough_texture`
