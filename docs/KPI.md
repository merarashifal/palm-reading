# Business Analytics & KPI Tracking

## Beta Launch Success Criteria

The beta is successful if the following loop executes seamlessly:
Upload ➔ 20s Analysis ➔ Free Report Loaded ➔ Curiosity Triggered ➔ Profile Captured ➔ Premium Unlocked ➔ PDF Downloaded ➔ Shared

## Funnel & Event Tracking

We track business events rather than technical files.

### 1. Engine & Analysis Events
- `upload_started`
- `quality_failed`
- `analysis_started`
- `analysis_completed`
- `analysis_failed`

### 2. User Engagement Events
- `report_opened`
- `report_read_25`
- `report_read_50`
- `report_read_75`
- `unlock_clicked`

### 3. Lead Capture Events
- `profile_started`
- `profile_completed`
- `premium_viewed`
- `pdf_downloaded`
- `shared`
- `returned_again`

## Report Status Lifecycle

Every report moves through these distinct states:
1. `Uploaded`
2. `Validated`
3. `Processing`
4. `Completed`
5. `Viewed`
6. `Unlocked`
7. `Downloaded`
8. `Shared`
9. `Archived`
